# Winget-Auto-Updater
Winget Auto-Updater is a lightweight Windows batch utility designed to automatically update all installed applications using Microsoft’s Winget package manager.
The script checks for administrator privileges, verifies Winget availability, then silently upgrades all eligible packages with minimal user input.

Go to [Releases](https://github.com/NotMathew/Winget-Auto-Updater.git) to download the script.
